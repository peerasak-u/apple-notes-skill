## [1.0.0] - 2024-01-XX

### Added
- Initial release of Apple Notes Skill
- Core commands: search, list, read, read-index, recent, create, delete
- HTML to Markdown conversion
- Markdown to HTML conversion
- Folder path support (nested folders)
- Unit tests for conversion functions
- Installation script
- Development tooling (linting, formatting, testing)

### Features
- Search notes by content
- List notes by title with indexing
- Read notes in Markdown format
- Create notes from Markdown
- Delete notes by title
- Get recently modified notes
- Support for nested folder paths
- Automatic HTML/Markdown conversion

### Documentation
- Comprehensive README
- Architecture documentation
- Contributing guidelines
- SKILL.md for OpenCode integration

### Testing
- Unit tests for HTML/Markdown conversion
- Unit tests for utility functions
- Test scripts for automation

### Tooling
- ESLint configuration
- Prettier configuration
- Node.js native test runner
- Installation script
